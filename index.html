⚙ Kết nối Google Sheets

Bước 1 — Tạo Google Sheet & Apps Script
1
Mở Google Sheets → tạo spreadsheet mới → đặt tên tùy ý
2
Vào menu Extensions → Apps Script
3
Xóa code cũ → paste toàn bộ nội dung file Code.gs vào → Save
4
Bấm Deploy → New deployment → Type: Web App → Execute as: Me → Who has access: Anyone → Deploy
5
Copy Web App URL → paste vào ô dưới đây
Bước 2 — Nhập Web App URL
URL sẽ được lưu trong trình duyệt. Chỉ bạn mới thấy URL này. Không ai có thể xem dữ liệu của bạn trừ khi họ có link Google Sheet.
